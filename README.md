DigiTools Platform

DigiTools Platform is a modern tools-based web application that provides a collection of useful digital utilities in a clean and user-friendly interface. It is designed to make everyday digital tasks easier and more efficient.

Live Demo
🔗 https://digitools-platform-modern-website.netlify.app/

GitHub Repository
🔗 https://github.com/probillal/digiTools-platform

✨ Features
 Collection of useful digital tools
 Fast and responsive UI
 Modern and clean design
 Fully mobile-responsive layout
 Easy navigation between tools

🛠️ Tech Stack
React.js
Tailwind CSS
JavaScript (ES6+)
React Router DOM

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/probillal/digiTools-platform.git

# Move to project directory
cd digiTools-platform

# Install dependencies
npm install

# Run the project
npm run dev
