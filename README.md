Name of Project: DigiTools Platform

Description:
DigiTools Platform is a user-friendly web application designed to provide multiple useful tools in a single interface. It focuses on improving efficiency, reducing repetitive work, and helping developers and users perform tasks quickly and smoothly.

The platform is built with a modern frontend approach and optimized for performance, responsiveness, and clean UI experience.

Technologies Used: React.js, Tailwind CSS, DaisyUI, REST API Integration, JavaScript (ES6+)

Key Features:

1. Dynamic Pricing card data loaded from API.
2. Add/remove item in cart.
3. Modern design with fully responsive layout for mobile, tablet, and desktop devices.
